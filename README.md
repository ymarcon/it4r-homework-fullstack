# ENAC-IT4R Homework Full Stack

## Setup

### Prerequisites

- [Make](https://www.gnu.org/software/make/)
- [Node.js](https://nodejs.org/) 18.x
- [Poetry](https://python-poetry.org/)
- [Python](https://www.python.org/) 3.10

### Installation

```bash
make install
```

### Development Run

```bash
make dev-backend
# http://127.0.0.1:8000

make dev-frontend
# http://127.0.0.1:5173
```
